# ElectroHub Shop

ElectroHub Shop is a demo electronics store built with **Next.js**, **Tailwind CSS**, and **Prisma**.  
It features a basic functional layout with Tailwind configured and a simple accent color.

## Features
- Next.js 16 with App Router
- Tailwind CSS 3.4.13 with custom accent color
- Basic layout with header, footer, and main content
- Globals CSS included
